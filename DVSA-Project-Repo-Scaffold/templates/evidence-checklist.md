# Evidence Checklist Per Lesson

For each lesson, try to keep these items:
- setup screenshot
- exploit request screenshot or terminal output
- backend proof screenshot (CloudWatch / IAM / AWS Console)
- before-fix code or policy screenshot
- after-fix code or policy screenshot
- post-fix verification screenshot
- one short text file with exact commands used

## Suggested per lesson folders
Each `evidence/lesson-XX/` folder can contain:
- `README.md`
- `commands.txt`
- `screenshots/` (optional if you want subfolders)
- `logs/` (optional)
